Important FAQ


What is the Document Object Model(DOM)?
The DOM is a tree-like structure that lets Javascript access and change the content of webpage.
How do you select an element from the DOM?
There are few ways to select an element from the DOM.For instance, getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll ect.
What is event delegation in the context of the DOM, and why is it useful?
ChatGPT said: Event delegation is a way to handle events (like clicks) using a single event listener on a parent element instead of adding listeners to each child element.
How do you manipulate an element's attribute and style using DOM?
To manipulate an element's attribute through the DOM, we can use:
setAttribute(name, value)- to add or change an attribute
removeAttribute(name)-to delete an attribute.
